# The Ledger & Bill Command Center (v5.3 Stable)

**The Ledger & Bill Command Center** is a lightweight, secure, and private personal finance tool designed to operate entirely within your web browser. It provides a comprehensive interface for tracking paychecks, managing a library of recurring and one-time bills, and visualizing spending habits without the need for an external server or account.

## üöÄ Key Features
* **Income Ledger:** Track paychecks with support for multiple granular deductions (Taxes, 401k, Health, etc.).
* **Bill Library:** Maintain a custom "Menu" of recurring and one-time expenses for fast data entry.
* **YTD Dashboard:** Automatic Year-to-Date calculation for Gross, Deductions, and Net income.
* **Visual Analytics:** Dynamic SVG Pie Chart for monthly expense distribution.
* **Data Portability:** * **Backup/Restore:** Save your entire database to a local `.json` file.
    * **CSV Export:** Export filtered monthly reports for use in Excel or Google Sheets.
* **Privacy First:** All data is stored in your browser's `localStorage`. No data is ever sent to a server.
